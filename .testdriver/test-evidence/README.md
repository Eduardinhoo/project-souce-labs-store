# Evidências de Teste

Esta pasta contém evidências da execução dos testes realizados no projeto de portfólio de Quality Assurance.

O objetivo deste diretório é armazenar capturas de tela (screenshots) e provas visuais da execução dos testes automatizados e manuais realizados durante o processo de validação do sistema.

## Conteúdo

* **screenshots/**
  Contém imagens capturadas durante a execução dos testes automatizados e a validação do comportamento do sistema.

## Exemplos de Evidências

Alguns exemplos de evidências que podem ser armazenadas nesta pasta:

* Execução bem-sucedida do teste de login
* Validação de login inválido
* Produto adicionado ao carrinho
* Validação do fluxo de checkout
* Resultado da execução dos testes no terminal com pytest

Essas evidências demonstram que os testes foram executados e ajudam a documentar o processo de testes realizado neste projeto.
